# Fortofolio
Repositori ini berisi proyek website portofolio pribadi yang aku buat untuk menampilkan karya dan kemampuan di bidang programming, desain, dan editing. Dibangun dengan gaya sederhana namun fungsional, situs ini jadi tempatku bereksperimen dengan ide-ide kreatif sekaligus menunjukkan hasil kerjaku secara lebih profesional dan santai.
